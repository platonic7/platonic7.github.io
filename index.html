API에서 알아두어야 할 클래스들

java.lang.String 문자열작업을 쉽게 할 수 있도록 제공되는 API 클래스

기존 언어들에서 char[ ]을 이용하는 방식 대신에 편리하게 사용할 수 있다는 장점

객체타입으로 사용한다는 특징

문자열의 생성 두 가지 상수처럼 손 쉬운 방식 String str = “AAA”; (이걸 많이 선호함) new를 활용하는 객체 방식 (이방식은 메모리상에 객체를 또 만들기에 사용 잘 안함) String str = new String(“AAA”);

두 방식의 메모리 활용 방식의 차이가 존재

‘==‘과 String의 equals( ) ‘==‘ 연산자 메모리상에 같은 공간을 가리키는 지를 확인하는 연산자

equals( ) 두 개의 String 객체가 같은 문자열을 내용으로 가지는지를 확인하는 연산자

public static void main(String[] args) {
    String str1 = "AAA";
    String str2 = "AAA";

    System.out.println(str1 == str2); //ture

    String str3 = new String("AAA");
    String str4 = new String("AAA");

    System.out.println(str3 == str4); // false

    boolean result = str3.equals(str4);

    System.out.println("equals: " + result); //true
}
